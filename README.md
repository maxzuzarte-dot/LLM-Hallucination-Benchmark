# LLM Hallucination Benchmark

## Overview

This project evaluates hallucination rates across leading Large Language Models.

Models evaluated:

- Claude
- GPT
- Gemini

## Research Question

Which model demonstrates the lowest hallucination rate on factual questions?

## Methodology

- 100 factual questions
- Manual verification
- Comparative evaluation

## Tools

- Python
- Pandas
- Matplotlib
